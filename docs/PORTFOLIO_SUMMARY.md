# I AM THE ONE™ SaaS v3 — Portfolio Summary

Built by Andrew Wolverton.

I AM THE ONE™ SaaS v3 is a full-stack commerce demo platform with a customer storefront and owner dashboard.

## What It Does

- Displays a premium customer storefront
- Shows live products with images
- Supports cart and checkout flow
- Creates real orders through a Flask API
- Stores order/product/setup request data in SQLite
- Provides an owner console for managing products and reviewing leads
- Captures setup request leads from potential buyers
- Runs locally and deploys through Render
