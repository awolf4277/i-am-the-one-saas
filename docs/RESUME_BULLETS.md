# Resume Bullets

- Built and deployed a full-stack SaaS storefront platform using React, TypeScript, Flask, SQLite, Render, and GitHub.
- Implemented customer storefront, cart, checkout/order creation, owner login, owner dashboard, product creation, and lead capture workflows.
