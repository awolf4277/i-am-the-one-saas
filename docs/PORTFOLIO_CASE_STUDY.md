# Portfolio Case Study

## Project

**I AM THE ONE™ SaaS v3 / WOLF OS™**

A full-stack storefront and owner dashboard system for small businesses, creators, and service providers.

## What I Built

I built a live SaaS-style commerce platform with a customer storefront, owner console, product creation, checkout/order flow, setup request lead capture, and deployment-ready documentation.

- Vite
- TypeScript
- Flask
- Python
- SQLite
- Render
- GitHub
- PowerShell

## Main Features

- Live product catalog
- Customer cart and checkout flow
- Owner login
- Owner dashboard
- Product creation from owner console
- Order tracking
- Setup request lead capture
- Buyer lead review inside owner console
- Production environment variables
- Render deployment
- Git tags and restore points
- Client setup documentation

## Problems I Solved

- Fixed frontend/backend API connection issues
- Configured CORS for Render deployment
- Fixed 405 Method Not Allowed errors by adding proper POST routes
- Added owner-protected product creation
- Built real setup request lead capture
- Solved stale Flask server issues during local testing
- Protected sensitive passwords from frontend exposure
- Added production/demo environment separation
